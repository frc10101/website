# Images Folder

This folder is for storing image assets for the CritHit website.

## Recommended Image Types

- **Hero Image**: Banner or gaming-related image for the hero section
- **Feature Icons**: Custom icons to replace emoji placeholders
- **Logo**: CritHit brand logo
- **Screenshots**: App or website screenshots for feature showcase

## Image Guidelines

- Use web-optimized formats (WebP, JPEG, PNG)
- Optimize file sizes for web performance
- Provide appropriate alt text in HTML
- Consider responsive images for different screen sizes
